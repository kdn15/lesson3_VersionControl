Highlight #1 Version Control

Version control records all changes to a file and keeps different versions of the file and makes sure that the files are organized. Version control is helpful to use because it keeps different versions of the same file incase you need to change something from a previous version. Also version control is useful in a group environment incase of people editing and changing the same file at the same time there will be different versions as well. Version control keeps notes of who changed the files, when they were changed, what about them is new or different, an being about to combine different versions of a file into one.

Highlight #2 Git

Git is a source code management system for software development. It uses a distributed revision control system. It emphasizes on speed, data integrity, and support for non-linear workflows. Git was designed and fully developed in 2005 by Linux kernel developers. Git is a free software because of the terms of the GNU General Public License version two. The developers of Git are Junio Hamano, Linus torvalds, and many others. The initial release date of Git was April 7, 2005. Git was written in C, Shell, Perl, Tci, and Python. Git is able to be used on Linux, Windows, and OS X.

Highlight #3 Remote Repository

Remote repository is a remote server that is shared among team members in a group. People are able to send their version of a file or receive a different version of the same file from the remote repository. For example, if I were to be working on a project in a group I would most likely have a different version than anyone else if I am working and changing the file. Then for me to share my version of the altered file I would push my version of the file to the remote repository for everyone else in the group to be able to pull the file.

Highlight #5 Github

Github is a website that has a Git repository on it and has distributed revision control and source code management. In 2015 Github reported that there are over eleven millions users and that there are twenty-nine point four million repositories. That being Github is the largest host of source code in the world. Github was founded on February 8, 2008. Its headquarters is in San Francisco, California. The founders of Github are Tom Preston-Werner, Chris Wanstrath, and PJ Hyett. The launch date of Github was April 10,2008. As of 2015 there are four hundred and fifteen employees working at Github.
